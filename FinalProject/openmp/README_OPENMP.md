# OpenMP Forward Convolution

This directory is flat by design. All files stay in this same directory.

## Files

- `conv_openmp.hpp` - function declaration
- `conv_openmp.cpp` - OpenMP convolution implementation
- `main_openmp.cpp` - file I/O, timing, matrix output
- `Makefile` - local build helper
- `openmp_job.sh` - Euler/SLURM job script

## Data files required

Before running, copy the same data generated by the sequential task:

```bash
cp ../seq/input.csv ./input.csv
cp ../seq/kernel.csv ./kernel.csv
```

The file sizes must match the dimensions in `openmp_job.sh`.

## Convolution model

For each PE/output pixel `(h,w)` and filter `co`:

```text
output[co][h][w] = sum input[ci][h+kh][w+kw] * kernel[co][ci][kh][kw]
```

Padding is only on the bottom and right boundaries.

## Run on Euler

```bash
sbatch openmp_job.sh
cat openmp_results.csv
ls openmp_filter_*.csv
```

## Manual run

```bash
make clean
make
export OMP_NUM_THREADS=20
./conv_openmp 64 64 3 8 3 5 input.csv kernel.csv 1 20 > openmp_results.csv
```
